# Используемые математические символы

В данном документе используются следующие нестандартные математические символы:

Символы теории множеств:
- ∅ - пустое множество
- ∩ - пересечение множеств
- ∪ - объединение множеств
- ∈ - принадлежит (элемент множества)
- ∉ - не принадлежит
- \ - разность множеств
- ⊆ - подмножество
- ⊂ - собственное подмножество
- ⋃ - объединение (для индексированных семейств)
- ⋂ - пересечение (для индексированных семейств)
- { } - фигурные скобки для обозначения множеств
- | - вертикальная черта (в конструкциях вида {x | условие})

Символы логики:
- ∀ - квантор всеобщности (для всех)
- ∃ - квантор существования (существует)
- ¬ - отрицание (не)
- ∧ - конъюнкция (и)
- ∨ - дизъюнкция (или)
- -> - импликация (если... то)
- ↔ - эквиваленция (тогда и только тогда)

Символы сравнения:
- ≤ - меньше или равно
- >= - больше или равно
- != - не равно
- < - меньше
- > - больше

Математические константы и переменные:
- λ (лямбда) - параметр распределения Пуассона
- δ (дельта малая) - минимальная степень вершины графа
- Δ (дельта большая) - максимальная степень вершины графа
- e - основание натурального логарифма
- π - число пи (не используется в данном документе, но может встречаться)

Математические операции:
- × - умножение
- · - умножение (альтернативное обозначение)
- ∏ - произведение (для последовательностей)
- ∝ - пропорционально
- ! - факториал
- ², ³, ⁴ и т.д. - степени
- ᵢ, ᵏ, ⁿ, ¹, ², ³, ⁴, ⁵, ⁶, ⁷, ⁸, ⁹, ¹⁰ - индексы и степени в нижнем/верхнем регистре

Обозначения множеств чисел:
- N - множество натуральных чисел
- Z - множество целых чисел
- R - множество действительных чисел
- Q - множество рациональных чисел (не используется в данном документе)
